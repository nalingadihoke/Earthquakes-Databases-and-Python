PREREQUISITES:

1. pandas, sys, mysql.connector
2. MySQL instance with a Hostname (example 'localhost'), Username (example 'root'), Password

EXECUTION:

Execution is the same for all the three scripts - Main_DB_Creation, Main_ETL, Main_Analysis

1. Open terminal
2. Change current directory to project folder
3. python [Filename].py <DB name> <hostname> <username> <password>

Note A - <DB name> is of user's choice - that is the database name that will be created.
Note B - /Data.Zip/ contains the data in cdv format - just in case!
